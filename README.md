# 👑 KingsQueen Downloader

A powerful and modern YouTube video downloader built using Flask and yt-dlp.  
Download videos in HD quality or extract audio as MP3 with real-time progress tracking.

---

## 🚀 Features

- 🎥 Download videos in 360p, 720p, 1080p
- 🎵 Convert and download MP3 audio
- ⚡ Fast and efficient downloads
- 📊 Real-time progress bar with speed & ETA
- 📱 Fully responsive (mobile + desktop)
- 🔒 Secure and clean UI
- 🧠 Smart format selection

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Downloader:** yt-dlp
- **Frontend:** HTML, CSS, JavaScript
- **Server:** Gunicorn
- **Deployment:** Render

---

## ⚙️ How It Works

Users paste a YouTube URL into the input field and the application fetches video details using yt-dlp.  
Available formats are displayed, allowing users to choose their preferred resolution or MP3 option.  
Once selected, the download starts in the background and real-time progress is shown until completion.
